1-氨基環丙烷-1-羧酸 (0 劃 / ACC)

同義詞： 1-aminocyclopropane-1-carboxylic acid
分類： 採收後處理-生理
是高等植物體內乙烯生合成路徑中,乙烯生成的立即前趨物質.由S-腺桥甲硫氨酸(S-adenosyl methionine, SAM)經由ACC合成酶催化生成.ACC經由ACC氧化酶氧化為乙烯.植物體內ACC含量多寡直接與其乙烯之生成量有密切關係.

來源：園藝科技術語/C90-2913.HTML#C90-2913 (作者：王自存)

ACC合成酶 (0 劃 / ACC synthase)

分類： 採收後處理-生理
高等植物體內乙烯生合成路徑中將SAM催化轉變為ACC之酵素.此一酵素在植物體乙烯生成之調控方面擔任重要角色,植物體之乙烯生合成,主要是由於ACC合成酶增加之結果,利用遺傳工程技術減少ACC合成酶的基因表現可以有效降低植物體之乙烯生合成量.

來源：園藝科技術語/C90-2913.HTML#C90-2920 (作者：王自存)

ACC氧化酶 (0 劃 / ACC oxidase)

同義詞： 乙烯形成酶 ethylene-forming enzyme EFE
分類： 採收後處理-生理
高等植物體內乙烯生合成路徑中將ACC催化轉變為乙烯之酵素,在1992年以前此一酵素被稱為乙烯形成酶.

來源：園藝科技術語/C90-2913.HTML#C90-2921 (作者：王自存)

α-氨基異丁酸 (0 劃 / α-amino isobutyric acid)

同義詞： AIB
分類： 採收後處理-生理
是一種ACC氧化酶抑制劑,此化合物之分子構造與ACC相近,可與ACC競爭ACC氧化酶上之結合部位,抑制酵素將ACC轉變為乙烯的反應速率.

來源：園藝科技術語/C90-2913.HTML#C90-2917 (作者：王自存)

乙烯 (1 劃 / ethylene)

分類： 採收後處理-生理 植物生理-植物生長調節劑
乙烯的分子式是C2H4,它是構造最簡單的烯類化合物,常溫中是氣態,為石油裂解的主要產物,也存在於有機物燃燒產生的煙中.乙烯也可由植物體自行生成,是一種重要的植物荷爾蒙,在植物生長發育過程中的某些階段,植物體或器官的乙烯生成量會漸漸升高,誘使植物體產生一些特定的生理反應.乙烯所影響的生理作用範圍很廣,如促進某些休眠種子的發芽,改變根、莖的生長,誘導鳳梨類植物的開花,促進葫蘆科瓜類的雌花形成,促進更年型果實的後熟及花朵的老化,促進葉綠素的分解與離層的產生.由於園產品在採收後的許多生理變化均與乙烯有關,因此如何控制乙烯是影響園產品採收後處理成敗的重要因素.

來源：園藝科技術語/C01-0001-1.html#C01-0007 (作者：王自存)

來源：https://agrovoc.fao.org/browse/agrovoc/en/page/c_2679

乙烯（英語：Ethylene）是由两个碳原子和四个氢原子组成的化合物。两个碳原子之间用双键连接。 乙烯為合成纖維、合成橡膠、合成塑料（聚乙烯及聚氯乙烯）、合成乙醇（酒精）的基本化工原料，也用於制造氯乙烯、苯乙烯、環氧乙烷、醋酸、乙醛、乙醇和炸藥等，且可用作水果和蔬菜的催熟剂，是一種已證實的植物激素，也是石油化工發展水準之指標。

來源：https://dbpedia.org/page/Ethylene

乙烯作用抑制劑 (1 劃 / ethylene action inhibitor)

分類： 採收後處理-技術
乙烯需要與細胞內特定的受體(receptor)結合後才能將其作用的訊息傳遞出去, 有些化合物能夠影響乙烯與其受體的結合關係,因而使乙烯的作用無法產生,這類化合物稱為乙烯作用抑制劑,其中最重要的一種是銀離子(Ag+),另外有2,5-正莰二烯(2,5-norbornadiene, NBD).

來源：園藝科技術語/C01-0001-1.html#C01-0011 (作者：王自存)

乙烯吸收劑 (1 劃 / ethylene absorbent)

分類： 採收後處理-技術
具有吸收並除去空氣中乙烯的製品.早期的吸收劑主要是含有溴的活性碳製品,目前最常見的乙烯吸收劑則是以過錳酸鉀為氧化劑的製品,以多孔性的蛭石、珍珠石或活性氧化鋁為吸附的物體.常用於園產品的冷藏及長距離運輸,以降低空氣中的乙烯含量.

來源：園藝科技術語/C01-0001-1.html#C01-0010 (作者：王自存)

乙烯生合成 (1 劃 / ethylene biosynthesis)

分類： 採收後處理-生理
高等植物自身具有合成乙烯的能力,其生化路徑為:甲硫胺酸(methionine)經由S-腺桥甲硫胺酸合成酶催化生成S-腺桥甲硫胺酸(SAM), SAM經由ACC合成酶催化生成1-氨基環丙烷-1-羧酸(ACC),最後ACC由ACC氧化酶氧化為乙烯.

來源：園藝科技術語/C01-0001-1.html#C01-0008 (作者：王自存)

乙烯生合成抑制劑 (1 劃 / ethylene synthesis inhibitor)

分類： 採收後處理-生理
具有抑制高等植物體內乙烯生成的化合物,這些物質主要抑制乙烯生合成路徑中相關酵素的活性,如AVG與AOA都是ACC合成酶的抑制劑,AIB是ACC氧化酶的抑制劑.

來源：園藝科技術語/C01-0001-1.html#C01-0009 (作者：王自存)

乙烯類似物 (1 劃 / ethylene analog)

分類： 採收後處理-生理
化學構造式與乙烯相似,而且也具有微弱乙烯生理作用的化合物,如丙烯(propylene, C3H6)、乙炔(acetylene, C2H2)、一氧化碳.

來源：園藝科技術語/C01-0001-1.html#C01-0012 (作者：王自存)

生理成熟 (5 劃 / physiological maturation)

生理障礙 (5 劃 / physiological disorder)

老化 (6 劃 / senescence)

同義詞： 衰老 衰化
分類： 植物生理-生長與發育 採收後處理-生理 
是植物體或其器官發育過程的最後一個階段,緊接在生理成熟之後,此時個體內部整體的合成作用速率逐漸低於整體的分解作用,細胞開始失去維持其基本生理作用的能力,個體會出現各種老化的徵狀,最後導致個體的死亡.園藝上有許多作物在未達到生理成熟即行採收,由於失去水分與養分的供應,在採收後亦進入老化階段.

是普通貯藏的一種方式,以山邊或高原上橫向挖鑿的山洞、隧道或土窯洞做為貯藏的場所,由於山洞內氣溫較恆定,受外界影響小,可以提供一個自然的貯藏環境.

來源：園藝科技術語/C15-2482.HTML#C15-2529 (作者：王自存)

選別 (15 劃 / sorting)

分類： 採收後處理-技術
將園產品依其成熟度、形狀、顏色或其他物理性狀加以區別分類,選別的方式可以分為人工選別及機械自動化選別.

來源：園藝科技術語/C16-2581.HTML#C16-2581 (作者：王自存)

16 劃

機械性傷害 (16 劃 / mechanical injury)

同義詞： physical injury
分類： 採收後處理-生理
指園產品因受到外力與實體接觸而造成的傷害,包括切割(cut)、擠壓(compression)、碰撞(impact)、摩擦(vibration)等,這些傷害會影響產品組織構造的完整,使得表皮破裂,細胞破損,進而造成生理與病理上的變化,使產品品質降低.機械性傷害是園產品採收後處理、運銷過程中最常發生的一種傷害.

來源：園藝科技術語/C16-2535.HTML#C16-2563 (作者：王自存)

輻照處理 (16 劃 / irradiation)

同義詞： 照射處理
分類： 採收後處理-技術 植物保護-有害生物防治
利用低劑量輻射線如伽瑪射線或X射線照射園產品的一種採收後處理手段.輻照處理的目的是做為一種滅蟲處理方法,或是抑制芽體的生長發育及果實後熟以延長產品的貨架壽命

來源：園藝科技術語/C16-2535.HTML#C16-2540 (作者：王自存)

17 劃

壓差預冷 (17 劃 / Pressure cooling)

分類： 採收後處理-技術
也稱作強制通風冷卻（Forced-air cooling），不同於室內風冷，其降溫的原理是利用抽風扇，在包裝紙箱的兩側製造壓力的差異，利用空氣會從壓力高處往壓力低處流動的原理，主動地使冷空氣由包裝箱一側的通風孔進入包裝箱中，直接與產品接觸後，由另一側通風孔流出，同時將箱內的熱帶走。

來源：https://kmweb.moa.gov.tw/subject/subject.php?id=42617 (作者：另查)

18 劃

燻蒸 (18 劃 / fumigation)

分類： 採收後處理-技術 植物保護-有害生物防治
將園產品放在密閉空間內,以氣態藥劑處理一段時間以殺死或抑制園產品所帶的害蟲、蟲卵及病原菌,常用的燻蒸劑有二溴乙烷(ethylene dibromide;EDB)、溴化甲烷(methyl bromide;MB)、氫氰酸(hydrocyanic acid;HCN)、二氧化硫(sulfur dioxide;SO2)等.

來源：園藝科技術語/C18-2707.HTML#C18-2725 (作者：王自存)

癒傷處理 (18 劃 / curing)

同義詞： 癒傷 療傷 調理 發汗
分類： 採收後處理-技術
指採收後,將產品放在一個特定的環境條件下一段時間,讓產品表面的傷口能夠迅速癒合或讓表皮乾枯形成保護層,以減少貯藏期間的失水及防止微生物入侵造成腐爛.

來源：園藝科技術語/C18-2744.HTML#C18-2790 (作者：王自存)

覆蓋堆藏 (18 劃 / clamp storage)

同義詞： covered heap
分類： 採收後處理-技術
是普通貯藏的一種方式,將產品堆放在地上,上面以稻草、麥稈、泥土或塑膠布等材料加以覆蓋,利用其底部簡單的通風管或以直接打開、蓋上覆蓋的方式來調節產品的溫度,使其維持在某一低溫範圍內以進行貯藏.

來源：園藝科技術語/C18-2707.HTML#C18-2711 (作者：王自存)

21 劃

露點 (21 劃 / dew point)

同義詞： 露點溫度 dew point temperature
分類： 採收後處理-技術
指空氣中的水蒸氣含量達到飽和程度時之溫度,即在此溫度時空氣之相對濕度為100%.當空氣溫度低於露點時,水氣會凝結成小水珠.

來源：園藝科技術語/C21-2847.HTML#C21-2856 (作者：王自存)
